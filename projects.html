Scatter
Voice-controlled personal assistant running on a Raspberry Pi 5. Custom-built from a bare OS installation. Whisper handles speech-to-text on-device. Smart model routing for language processing — lightweight model for fast queries, heavier model for complex reasoning. Persistent memory via SQLite. Integrated with Gmail, Google Calendar, Home Assistant, and Tavily for real-time search.

Adaptive K-12 education platform (3K–12th grade).
